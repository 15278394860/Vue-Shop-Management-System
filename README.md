# Vue-Shop-Management-System

## 项目简介

***

电商管理系统，使用vue+elementUI，实现的功能有用户管理，权限管理，分类管理，参数管理，商品管理，数据统计。仓库内容包含源代码+个人总结笔记+参考资料
